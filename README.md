# Capstone 1

EMS Console App: capstone1.dart as main file
and capstone1dependency.dart as imported package.

## Overview

main page:

- login
- sign up
- admin contact
- exit

admin functions:
- add users
- add salary
- add leaves
- view users
- salary report
- manage leaves

user functions:
- view and edit profile
- request a leave
- view active leaves
